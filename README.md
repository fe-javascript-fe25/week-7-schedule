# Schema, vecka 7
###### JavaScript, vecka 6 av 8

## Introduktion

I den här modulen är det äntligen dags att ta oss an API-anrop – alltså asynkrona anrop där vi hämtar data från externa resurser. 
Vi kommer också att fördjupa oss i det som inom JavaScript kallas moduler, och lära oss hur vi på ett snyggt och strukturerat sätt kan dela upp vår kod i flera filer.

## Mål för veckan:

1. Förstå vad det innebär att en funktion är asynkron
2. Förstå vad ett API är, samt hur vi kan använda det för att hämta data från webben
3. Förstå hur vi kan dela upp vår kod i olika filer och funktioner
